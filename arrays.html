Создание массивов NumPy

Массивы NumPy можно создавать с помощью различных методов, таких как:

  • np.array([1, 2, 3])
  • np.zeros((2, 2))
  • np.ones((3, 3))
  • np.arange(0, 10, 2)

Атрибуты массива

Массивы NumPy имеют несколько важных атрибутов:

  • ndarray.shape: Возвращает размеры массива.
  • ndarray.dtype: Возвращает тип данных элементов массива.
  • ndarray.size: Возвращает общее количество элементов в массиве.
