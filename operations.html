Основные операции

NumPy позволяет выполнять поэлементные операции:

  • np.add(array1, array2): Складывает два массива поэлементно.
  • np.subtract(array1, array2): Вычитает два массива поэлементно.
  • np.multiply(array1, array2): Умножает два массива поэлементно.
  • np.divide(array1, array2): Делит два массива поэлементно.

Математические функции

NumPy предоставляет широкий набор математических функций:

  • np.sqrt(array): Вычисляет квадратный корень каждого элемента.
  • np.sin(array): Вычисляет синус каждого элемента.
  • np.log(array): Вычисляет натуральный логарифм каждого элемента.
